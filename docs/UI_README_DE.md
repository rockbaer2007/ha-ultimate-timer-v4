# 🖥️ UI (Mushroom Dashboard)

## 🧠 Features der UI

👉 deutlich moderner als Standard:

- 🧱 **Mushroom Cards** (clean & modern)  
- ▶️ **Start / Stop als Buttons**  
- 🔄 **Status als farbige Cards**  
- ⏳ **Countdown zentral angezeigt**  
- 🎨 **Automatische Farben** (grün = aktiv)  
- 📱 **Perfekt für Mobile**

---

## ⚠️ Voraussetzung

👉 du brauchst:

- **Mushroom Cards installiert**

Installation über:

- HACS → Frontend  
- 👉 **Mushroom**

---

## 🚀 Verwendung

1. Mushroom über HACS installieren  
2. YAML Dashboard importieren  
3. Entitäten anpassen (z.B. pool_timer_*)  
4. Fertig 😎

---

## 💡 Hinweis

Diese UI ist optional und dient als moderne Visualisierung für den Ultimate Timer V4.
