# 🖥️ UI (Mushroom Dashboard)

## 🧠 UI Features

👉 much more modern than the default:

- 🧱 **Mushroom Cards** (clean & modern)  
- ▶️ **Start / Stop as buttons**  
- 🔄 **Status as colored cards**  
- ⏳ **Countdown displayed centrally**  
- 🎨 **Automatic colors** (green = active)  
- 📱 **Perfect for mobile devices**

---

## ⚠️ Requirements

👉 you need:

- **Mushroom Cards installed**

Install via:

- HACS → Frontend  
- 👉 **Mushroom**

---

## 🚀 Usage

1. Install Mushroom via HACS  
2. Import the YAML dashboard  
3. Adjust entities (e.g. pool_timer_*)  
4. Done 😎

---

## 💡 Note

This UI is optional and provides a modern visualization for the Ultimate Timer V4.
