# Ultimate Timer V4

Advanced Home Assistant timer with:
- HH + MM input
- Stable blueprint (no template errors)
- Mushroom UI included
- Countdown support

## Features
- Start / Stop control
- Hours + Minutes input
- Presets
- Works with helpers

## Requirements
- Mushroom Cards
- input_number helpers (hours + minutes)
- input_boolean (start/stop/running/done)

## Installation
1. Copy blueprint
2. Import into HA
3. Create helpers
4. Use UI
